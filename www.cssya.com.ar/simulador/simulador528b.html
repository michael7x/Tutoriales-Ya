9 - Herencia de propiedades de estilo.




Problema:

Confeccionar una página aplicando el siguiente estilo:

body {
  color:#0000ff;
  font-family:verdana;
}

Luego imprimir contenido dentro de los elementos h1,em y p.

Definir dos reglas más de estilo al problema, que sobreescriban la propiedad color:

em {
  color:#008800;
}
p {
  color:#999999;
}
