3 - Definición de estilos a nivel de página.




Problema:Definir un estilo diferente para los elementos h1,h2,h3,h4,h5 y h6. Mostrar mensajes utilizando estos elementos a las que se le han definido estilos (por ahora solo conocemos el color de texto y fondo) 
<!DOCTYPE html>
<html>
<head>
<title>Prueba de CSS</title>

<style>
h1 {color:##ff0000;background-color:#cccccc}
h2 {color:##ff0000;background-color:#bbbbbb}
h3 {color:##ff0000;background-color:#aaaaaa}
h4 {color:##ff0000;background-color:#999999}
h5 {color:##ff0000;background-color:#888888}
h6 {color:##ff0000;background-color:#777777}
</style>

</head>
<body>

<h1>Titulo de nivel 1</h1>
<h2>Titulo de nivel 2</h2>
<h3>Titulo de nivel 3</h3>
<h4>Titulo de nivel 4</h4>
<h5>Titulo de nivel 5</h5>
<h6>Titulo de nivel 6</h6>

</body>
</html>
