El objetivo de este tutorial es poder aprender HTML en forma sencilla viendo un concepto teórico, luego algunos ejercicios resueltos y por último y lo más importante desarrollar otros por usted mismo.
Puede desarrollar los ejercicios en el sitio, probarlos y ver los resultados inmediatamente.
A lo largo del curso se introducen conceptos de HTML desde CERO, es decir no se requieren conceptos previos.

Luego de desarrollar el curso de HTML puede visitar los avances actuales de HTML 5.
Puede visitar también los tutoriales de: CSS y el framework de CSS más popular Bootstrap.
